Projet de programmation orienté objet

Réalisation d'un jeu vidéo sur le thème du jeu Hearthstone - Janvier 2023

Dans le cadre du cours de programmation orienté objet, nous avons eu pour projet de réaliser un jeu vidéo sur le thème du jeu Hearthstone.

Nous avons utilisé le moteur de jeu Unity pour la partie graphique et le langage C# pour la partie programmation.

Le jeu est un jeu de carte en tour par tour où le but est de réduire les points de vie de l'adversaire à 0.

Projet de conception d'application

Réalisation d'une carte web interactive - Février 2024

Durant le cours de conception d'application, nous avons eu pour projet de réaliser une carte web interactive.

Nous avons utilisé le framework React pour la partie front-end et le framework ASP.NET pour la partie back-end.

La carte permet de mesurer la distance entre un point et les villes les plus proches. Des filtres peuvent être appliqués pour affiner la recherche.

Projet de programmation GPU

Rendu de fumée en temps réel - Avril 2024

Dans le cadre du cours de programmation GPU, nous avons eu pour projet de réaliser une simulation de fumée en temps réel.

Nous avons utilisé le moteur de jeu Unity pour la partie graphique et le langage C# pour la partie programmation.

La simulation est basée sur un système de particules qui se déplacent de façon aléatoires. La fumée est générée à partir de ces particules.

Les calculs de rendu sont effectués sur la carte graphique.
